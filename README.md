# Procesamiento_Lotes
Simulación de Procesamiento por Lotes (Python)

Este proyecto es una simulación básica del procesamiento por lotes, donde varios procesos se capturan desde el teclado y se agrupan en lotes de 4.
Luego, cada proceso se ejecuta de manera secuencial mostrando:
- Tiempo transcurrido
- Tiempo restante
 -Operación realizada
- Resultado
- Contador global
- Procesos terminados
- Lotes pendientes
Es un proyecto académico diseñado para practicar estructuras de datos, ciclos, funciones y simulación de tiempo en Python.

# Funcionalidades
Captura interactiva de procesos desde teclado.
Validación de:
- Operación válida.
- Tiempo estimado mayor a 0.
- Número de programa único.
- Agrupación automática en lotes de máximo 4 procesos.
- Simulación realista del tiempo con time.sleep().
- Cálculo automático de la operación asignada al proceso.
- Registro de procesos terminados.
- Contador global de tiempo.
Visualización del estado del sistema:
  - Lotes pendientes
  - Lote en ejecución
  - Procesos terminados
  - Tiempos de ejecución
